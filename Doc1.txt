:writing*hand: \*\*\_Sporting Event System [SES]*\*\*  
SES is an application which allows users to manage various sporting events.
Users can:
Manage teams and players
Manage tournaments and matches
Manage fields 
Manage tickets

Must Have -- 
Tournament, Match Name, 
Date, Time, Price, Book Tickets, Payment Type , Location 
Different prices (VIP, Standard), Seat Map 
  
Should Have -- Login

Could Have -- Live Stream On Website , Promotion 

Will Not Have -- advertisments, multiple logins
